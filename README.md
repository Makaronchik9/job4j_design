# job4j_design

## Описание проекта

`job4j_design` — это учебный репозиторий, содержащий решения задач из курса **Job4j Джуниор**, посвящённого структурам данных, вводу-выводу, работе с базами данных, чистой архитектуре и алгоритмам.
